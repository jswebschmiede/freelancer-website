## Merkmale der Webseite

Im Rahmen dieses Projekts wurde eine individuell gestaltete Webseite für Herrn Marcel Kapelle konzipiert. Das Hauptziel war es, eine ästhetisch ansprechende Online-Präsenz zu schaffen, die seine Dienstleistungen in bestem Licht zeigt und gleichzeitig seinen Kunden eine benutzerfreundliche Plattform bietet.

Kunden können schnell und einfach wichtige Informationen wie Öffnungszeiten, Services und Kontaktinformationen finden.

### Statische Webseite mit Astro und Tailwind CSS

Die Webseite wurde mithilfe von Astro und Tailwind CSS entwickelt. Astro, ein modernes Framework für statische Seiten, welches eine komponentenbasierter Entwicklung ermöglicht. Tailwind CSS, ein leistungsfähiges Utility-First-CSS-Framework, diese wurde eingesetzt, um eine maßgeschneiderte und dennoch konsistente Benutzeroberfläche zu erstellen.

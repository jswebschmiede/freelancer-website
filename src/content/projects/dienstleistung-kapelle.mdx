## Merkmale der Webseite

Ich habe für Marcel Kapelle eine Webseite entwickelt, die seine Dienstleistungen für Haus und Garten klar vorstellt. Der Auftritt sollte Leistungen, Kontaktwege und die wichtigsten Informationen ohne unnötige Umwege vermitteln.

Besucher finden dort die angebotenen Services sowie Kontaktinformationen in einer übersichtlichen Struktur.

### Statische Webseite mit Astro und Tailwind CSS

Die Webseite wurde mit Astro und Tailwind CSS umgesetzt. Astro eignet sich hier für einen schlanken statischen Aufbau, während Tailwind CSS eine saubere und durchgängige Gestaltung der Oberfläche unterstützt.
